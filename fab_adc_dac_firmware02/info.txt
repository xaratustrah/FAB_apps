Firmware Application for FAB_ADC/DAC_Rev.B Board
------------------------------------------------

27.04.2007/sh

This folder contains the main application written for the CPLD on the 
FAB-ADC/DAC Board Rev.B

It implements the main register file for the CPLD which contains the 
functionalities of the board. The firmware follows the TADI concept.


11.02.2009/sh

fib_adcdac_dac_fcn_gen1
